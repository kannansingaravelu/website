timeseries.cagr(returns: Series, intra_period: int = 1, is_log: bool = False) → float

Compounded Annual Growth Rate (CAGR) is the annual rate of return realised by an asset or portfolio to reach its current market value from its initival value.

CAGR is a return measure that provides ease of comparison between different trading strategies. However, this should always be used in conjunction with a risk measure such as volatility or standard deviation.

CAGR = (Ending Value / Starting Value)^(1/n) - 1

Input

Returns : Series of returns Period : intra_period = 1 for annual timeframe ReturnType : defaults to ‘False’ if its simple return

Output

Returns CAGR for the specified period

timeseries.calmar(returns: Series, is_log: bool = False) → float

Ratio of compounded annual growth rate and maximum drawdown. It is a measure of risk adjusted return. Lower the ratio, the worse the performance on a risk-adjusted basis.

Calmar Ratio = CAGR / MDD

Input

Returns : Series of returns ReturnType : defaults to ‘False’ if its simple return

Output

Returns calmar ratio

timeseries.maxdd(returns: Series, is_log: bool = False) → float

A maximum drawdown (MDD) is an indicator of downside risk and measures the largest percentage drop of the cumulative return over a specified time period.

It observes the maximum loss from a peak to a trough of a portfolio before a new peak is attained.

MDD = (Peak Value - Lowest Value) - Peak Value

Input

Returns : Series of returns ReturnType : defaults to ‘False’ if its simple return

Output

Returns MDD for the specified period

timeseries.sharpe(returns: Series, is_log: bool = False, rf: float = 0.0) → float

Sharpe ratio is the average return earned in excess of the risk free return for every unit of volatility. This is one of the most widely used meausre of risk adjusted return. Sharpe ration greater than 1 is considered to be good.

Sharpe Ratio = (Expected Return - RiskFree Return) / Volatility of Returns

Input

Returns : Series of returns ReturnType : defaults to ‘False’ if its simple return RiskFreeRate: RiskFree rate of return, defaults to 0.

Output

Returns sharpe ratio

timeseries.volatility(returns: Series, intra_period: int = 1) → float

Annualized volatility or standard deviation of returns is key risk metrics of any trading strategy as this captures the variability of returns from the mean return.

Annualization is achieved by multiplying volatility with square root of a) 252 to annualize daily volatility b) 52 to annualize weekly volatility c) 12 to annualize monthly volatility

Input

Returns : Series of returns Period : intra_period = 1 for annual timeframe

Output

Returns annualized standard deviation or volatility
