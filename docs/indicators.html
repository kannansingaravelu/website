indicators.ATR(df: DataFrame, lookback: int = 14) → Series

Average True Range (ATR) is a volatility indicator that measures n-period average of the true range (TR) values.

An expanding ATR indicates increased range of the underlying, while a low ATR value indicates a series of periods with small ranges (quiet days).

Input

DataFrame : DataFrame with OHLC columns Lookback : Lookback period for averaging

Output

Returns ATR for the specified period
